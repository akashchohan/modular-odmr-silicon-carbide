# Images & Setup Photos

This folder contains high-resolution photos documenting the construction and experimental setup of the ODMR system.

Subfolders:
- `rf_synthesizer/` — Si5351 board, wiring, GUI screenshots
- `adf4351/` — Clock generator module and setup
- `laser_setup/` — 808 nm laser diode, optics, TEC cooling
- `red_pitaya/` — RP connections and oscilloscope setup
- `optics_alignment/` — Beam alignment using green laser
- `final_experiment/` — Complete assembled system

These images serve as build proof and experimental evidence.
